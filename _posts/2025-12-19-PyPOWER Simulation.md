_Eudoxys Sciences releases `fips` developer tool to access US Census Bureau FIPS data_.

The `pypower_sim` package provides a quasi-steady time-series solver for [pypower](https://www.eudoxys.com/pypower_sim). The simulation supports geographic inputs and outputs using the [geohash](https://www.eudoxys.com/geohash), [loads](https://www.eudoxys.com/loads), and [eia](https://www.eudoxys.com/eia) packages from Eudoxys.

For more information, see the [Eudoxys `pypower_sim` documentation](https://www.eudoxys.com/eudoxys/pypower_sim).
