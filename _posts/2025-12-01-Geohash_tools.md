_Eudoxys Sciences releases `geohash` developer tool to index utility assets using geographic hash codes_.

The `geohash` tool converts latitude/longitude data into geographic hash codes that can be used for indexing. The hash codes follow the [CTA-5009](https://shop.cta.tech/products/cta-5009) standard.  

For more information, see the [Eudoxys `geohash` repository on github](https://github.com/eudoxys/geohash).
